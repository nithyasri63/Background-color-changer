# Background-color-changer
A simple Background Color Changer built using HTML,CSS and JavaScript.Users can change the background color by clicking color buttons and the selected color is saved using localStorage.

##Features

  Change background color with one click.

  Display the current selected color.

  Save selected color using Local Storage.

  Responsive design for Mobile and Desktop.


##Technologies Used

  HTML5

  CSS3

  JavaScript
